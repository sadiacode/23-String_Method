# String Methods in Python
## Description
This project is a simple python program that demonstrates how to use string methods in python. String methods are built-in functions that help modify,search,, and manipulate strings. 
## String Methods
**Methods**                     **Description**
- lower():                       Converts to lowercase
- upper():                       Converts a uppercase
- capitalize():                   Capitalizes only the first word
- title():                       Capitalizes the first letter of each word
- count():                        Count occurences
- find():                         Returns the index of first occurence
- replace():                      Replaces part of string
- startswith():                  Check if the string starts with given text
- endswith():                       Check if the string ends with given text
